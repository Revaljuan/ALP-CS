ALP - Demo Sistem Otorisasi

Aplikasi REST API dengan otorisasi modern: JWT, OAuth2, RBAC & ABAC

API Endpoints

  • POST /api/login - Body: {"username":"...", "password":"..."}
  • GET /api/documents - Header: Authorization: Bearer <token>
  • POST /api/documents - Create document
  • DELETE /api/documents/<id> - Delete document
  • Google OAuth: /auth/login
